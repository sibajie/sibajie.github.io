凌波多媒体教学网的安装和卸载

发表于 2016-08-23 更新于 2021-07-14

安装前的准备:

1.准备安装

安装凌波多媒体教学网软件前,请先检查您的系统是否符合运行需求,以及TCP/IP网络协议是否设置好.

2.系统需求

首先,您应检查您的计算机系统是否符合凌波多媒体教学网软件的运行条件.下表列出了系统要求:
处理器 奔腾133 或更高处理器.
内存 32 MB 或以上.
显卡 Windows 支持的 100% VGA 兼容图形适配器.
磁盘空间 各模块所需磁盘空间均小于 10 MB.
操作系统 Windows XP、Windows 2000、Windows NT 4.0、Windows Me、Windows 98、Windows 95.
网络 10/100M或以上.
如果操作系统是 Windows NT 4.0,需另外安装WMI、Microsoft Service Pack 4 或以上版本.
如果操作系统是Windows 98 或其OSR版本,需另外安装WMI、DirectX 7.0或以上版本.
如果操作系统是Windows 95 或其OSR版本,需另外安装WMI、DirectX 7.0、DirectMedia 6.0、WinSock2或以上版本.
如果使用语音功能,还需要有全双工声卡.

3.网络协议

运行凌波多媒体教学网需要TCP/IP协议的支持.如果网络中没有服务器或服务器上没有安装DHCP服务,此时网络中所有计算机的网卡所绑定的TCP/IP协议的设置上必须指定一个固定的IP地址.如果网络中有Windows NT 4.0 或 Windows 2000 服务器,并且服务器上安装有DHCP服务,此时学生机网卡所绑定的TCP/IP协议的设置上可以设为自动获取IP地址.具体的设置请咨询您所使用的网络的网络管理员.
开始安装.
凌波多媒体教学网程序由两部分构成:教师端程序和学生端程序,教师端程序安装在一台教师用电脑上,学生端程序安装在多台学生用电脑上.这些电脑用TCP/IP协议构成一个局域网.

1.教师机的安装

1)如果以前安装的教师端程序正在运行,首先要把教师端程序关闭;

2)执行Setup.exe程序进行安装,如果系统中已经安装过凌波多媒体教学网软件,系统会提示是否删除以前安装的程序,这时可按提示把以前已经安装的删除掉,然后重新运行Setup.exe,开始安装;

3)按照安装提示选择安装部件:教师端;

4)按照安装提示完成安装.

2.学生机的安装

1)如果以前安装的学生端程序正在运行,首先要把学生端程序关闭(可以用热键Ctrl+Win+F12把学生端程序关闭);

2)执行Setup.exe程序进行安装,如果系统中已经安装过凌波多媒体教学网软件,系统会提示是否删除以前安装的程序,这时可按提示把以前已经安装的删除掉,然后重新运行Setup.exe,开始安装;

3)按照安装提示选择安装部件:学生端;

4)按照安装提示完成安装.

3.教师机的卸载

用[控制面版]中的[添加删除程序],可以把教师端程序删除.

4.学生机的卸载

用[控制面版]中的[添加删除程序],可以把学生端程序删除,注意,删除学生端程序时,首先要把学生端程序关闭,可以用热键Ctrl+Win+F12把学生端程序关闭.
