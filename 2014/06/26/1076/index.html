发表于 2014-06-26 更新于 2021-07-14 分类于 各类服务

今天我们来学习一下NFS的文件同步系统,首先介绍一下NFS:NFS是Network File System的简写,
即网络文件系统.网络文件系统是FreeBSD支持的文件系统中的一种,也被称为NFS.
NFS允许一个系统在网络上与他人共享目录和文件.通过使用NFS,
用户和程序可以像访问本地文件一样访问远端系统上的文件.
